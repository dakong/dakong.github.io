Undergraduate Business Network

Technology used: HTML5, CSS3, Bootstrap Framework, JQuery, Angular JS

Built a full stack website for the Undergraduate Business Network. The idea was to create a website that had a clean professional user interface that displayed information about the student organization. Some functionalities includea contact form along with an integrated Google map to the UCSD campus. Angular JS was used to show the member profile pictures on the members page.
